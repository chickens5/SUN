🔧 Getting Started

  1. The notebook loads NASA OMNIweb solar wind data (1,300+ samples)
  2. Trains a Gradient Boosting model on historical Kp index labels
  3. Generates 3-day future scenarios (Quiet, Moderate, Active)
  4. Predicts solar wind parameters and Kp index
  5. Calculates storm risk metrics for each scenario

Key Physics: Newell coupling function relates solar wind conditions to magnetosphere energy input:

ε = V^(4/3) × B_t^(2/3) × sin^(8/3)(θ/2)

Where V = speed, B_t = total field, θ = clock angle

📚 Algorithm Overview

The model uses three key components:

  1. Feature Engineering: Newell coupling, rolling statistics, and solar wind parameters
  2. Model Training: Gradient Boosting with cross-validation on multi-year data
  3. Ensemble Forecasting: Three physics-grounded scenarios (quiet→moderate→active)
