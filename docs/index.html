Welcome!

This is the landing page for the geomagnetic storm predictor results based off the GGSP notebook. We're currently static on GitHub Pages, so this page serves as both an introduction to the notebook and a guide to how the static implementation works and can be published.

What this notebook does

The Geomagnetic Storm Predictor ingests live solar wind data, trains a model on historical observations, and forecasts planetary Kp for the next 72 hours.

Data window
NOAA 7-day + OMNI multi-year
Model
Gradient boosted regression
Forecast
3-day scenario ensemble
01 / Ingestion

Live solar wind streams

The notebook pulls NOAA plasma, magnetic field, and planetary Kp feeds, then falls back to a synthetic but physically plausible dataset when the network is unavailable.

02 / Modeling

Chronological training

Multi-year OMNI data provides a stronger time-series split and a more trustworthy error profile than a tiny random slice of one week.

03 / Forecasting

Scenario ensemble

Quiet, moderate, and active futures are projected for the next 72 hours so the notebook can communicate risk instead of just emitting a single point estimate.
